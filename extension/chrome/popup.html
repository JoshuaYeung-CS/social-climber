Set min = max for a fixed cadence. A different range (e.g. 15–30 min) makes each run land at a random time inside that window so the pattern looks less robotic to Instagram.

Opens favorited accounts that haven't been archived yet, one at a time, paced so IG doesn't rate-limit. Skips accounts whose media folder you emptied — that's your "I cleared this on purpose" signal. Requires "Auto-archive viewed posts & reels" enabled below.
