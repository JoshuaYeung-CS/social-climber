Run export

Opens Meta's data-export page and auto-fills it with your saved preferences (JSON, followers + following, all time, Google Drive). You handle the password if it asks.

–

Set min = max for a fixed cadence. A different range (e.g. 15–30 min) makes each run land at a random time inside that window so the pattern looks less robotic to Instagram.

Tools

Failure alerts (push to phone on 2+ consecutive failures) are configured server-side at ~/.config/igtracker/push.json. iMessage to your own number is recommended — no third-party app, E2E by Apple. See the server /api/push docs.
